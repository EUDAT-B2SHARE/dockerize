1. Download the dockerize repository	
2. Export the necessary env variables	
3. Run `docker-compose up`	
4. Go to localhost:80 and login	
5. Create an application token from your profile page	
6. Copy it and run the testing script with ./test_b2share_upgrade.sh $ACCESS_TOKEN	
7. After it finishes run `docker-compose down`	
"8. Edit the docker-compose.yml to use the b2share v2.1.0 image: 
                                   ...
                                   b2share:
                                        image: eudatb2share/b2share:2.0.1
                                   ..."	
9. Run `docker-compose up`	
10. Run the test_b2share_upgrade_2.sh $ACCESS_TOKEN
